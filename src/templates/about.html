Esta pagina web fue creada como un proyecto para la materia de Taller de Base de Datos, impartida por la profesora Elizabeth Cortez Razo, en el Instituto Tecnologico de Ciudad Madero

Fue creada por los estudiantes:

  • Alva Segundo Samuel
  • Espitia Ramirez Giancarlo
  • Garcia Mendoza Rafael

Fue creada utilizando como servidor Node.js junto a MySQL como motor de base de datos, el servidor utiliza como framework web Express.js, las paginas dinamicas son renderizadas utilizando un sistema de plantillas, en este caso Nunjucks.

Los estilos usados son parte del proyecto Tailwindcss.
